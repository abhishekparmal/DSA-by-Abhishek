# DSA-by-Abhishek

Welcome to **DSA-by-Abhishek**! This repository contains implementations of essential **Data Structures and Algorithms (DSA)** concepts in **Java** and **JavaScript**.

- Linked Lists

- Sorting & Searching
- Hashing
- Trees & Graphs
- Dynamic Programming
- Greedy Algorithms
- Bit Manipulation
